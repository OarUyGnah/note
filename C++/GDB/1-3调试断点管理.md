### 设置断点

- break/b

  ```bash
  b main		  	# main函数断点
  b main.cpp 38	# 38行断点
  b func 		 	# 名字为func的所有函数都打断点
  ```

- rb正则表达式

  有的函数为work有的为test_work，都想打断点

  ```bash
  rb work
  ```

- 条件断点

  ```bash
  b main.cpp:14 if i == 90 # 当14行i == 90时才为断点
  ```

- 临时断点tb

  只会命中一次

### 查看断点

```bash
i b
info breakpoints
info break
```

### 删除断点

```
delete 断点Num
```

### 禁用/启用断点

```bash
disable 断点Num
enable 断点Num
```

### 保存/载入断点

将当前断点保存在文件中，节省下次打断点的过程

```
save breakpoints filename
source filename
```
